Earn Crypto

Earn by Being a Node

Turn your unused hard drive space into passive income. Download the node software, choose how much space to rent out, and start earning automatically based on AI reputation scores.

Windows Desktop or Server

1. Download the Windows EXE

‚¨á Download neuro-node.exe

2. Double click the `.exe` file. The terminal will ask you how much storage to rent in GB.

3. Keep it running in the background to earn!

macOS (Intel & Apple Silicon)

1. Download the macOS Executable

‚¨á Download neuro-node (macOS)

2. Open your Terminal and run the downloaded file.

cd ~/Downloads
chmod +x neuro-node-macos
./neuro-node-macos

3. Enter your storage amount when prompted.

Linux (Ubuntu, Debian, CentOS)

1. Run this one-line installer in your terminal:

curl -sL https://neurostore.network/install.sh | bash

2. Run the node directly:

neuro-node

3. The CLI will ask you how much storage to rent. To daemonize it, we provide a systemd template.

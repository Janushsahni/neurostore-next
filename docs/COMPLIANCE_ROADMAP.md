# Compliance Roadmap

## Phase 1: MVP Testnet
- baseline audit logs for auth, data operations, and payouts
- privacy policy and data processing addendum draft
- encryption and key ownership model documented

## Phase 2: Paid Beta
- GDPR controls:
  - data inventory and retention policy
  - deletion workflow with audit log
  - region-aware placement controls
- SOC 2 Type I readiness:
  - access control policy
  - change management policy
  - incident response runbook

## Phase 3: Mainnet GA
- SOC 2 Type I assessment complete
- start SOC 2 Type II evidence collection window
- formal vendor risk management process

## Phase 4: Enterprise Expansion
- HIPAA-ready optional tier:
  - BAA process
  - stricter tenant isolation model
  - PHI-specific access logging
- customer-managed keys and dedicated compliance reports
